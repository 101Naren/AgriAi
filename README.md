# AgriAi
Building my own AI chatbot. This is an idea which was developed at Make-A-thon 2025. The problem statement: Build an AI based query system for farmers.
Features:
  1) Agriculture-focused AI assistant
  2) Context based prompt engineering
  3) Uses Llama 3.2 via Ollama
  4) REST API endpoint for chat
  5) CORS enabled for frontend
  6) Windows compatiable Ollama command handling
  7) Language aware responses
Tech Stack:
  1) Python
  2) Flask
  3) Flask-CORS
  4) Ollama
